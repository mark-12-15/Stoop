## Design Specifications

### Main Logo
- **Icon**: 3 stacked horizontal bars (decreasing width from bottom to top)
- **Checkmark**: Floating above the bars
- **Colors**: 
  - Primary blue: `#1E40AF`
  - Accent green: `#10B981`
- **Text**: "Stoop" (blue) + "Keep" (green)

### Usage Guidelines

#### Main Logo (stoopkeep-logo-horizontal.png)
- Use on: White/light backgrounds
- Navigation bars, landing page, marketing materials
- Maintain minimum width: 120px for legibility

#### Dark Mode Logo (stoopkeep-logo-horizontal-dark.png)
- Use on: Dark backgrounds (#111827 or similar)
- Dark mode website, dark presentations
- Icon and "Stoop" text: White (#FFFFFF)
- "Keep" text: Light green (#34D399)

#### Monochrome Logo (stoopkeep-logo-horizontal-monochrome.png)
- Use for: Single-color printing, embroidery, stamps
- All elements: Deep blue (#1E40AF)
- "Keep" uses bolder font weight for differentiation

#### App Icon (stoopkeep-icon-512.png)
- Use for: App icon, favicon base, social media profile
- Square format with centered icon
- No text, icon only
- Padding: 15-20% on all sides

## Brand Guidelines

1. **Do not** distort or stretch the logo
2. **Do not** change colors (except for approved variants)
3. **Do not** add effects (shadows, outlines, glows)
4. **Maintain** minimum clear space around logo (equal to height of one bar)
5. **Use** provided color values exactly
